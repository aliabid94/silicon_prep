Classes

Silcon Prep classes are all held online with a live instructor. Students are provided a link where they can watch the instructor, ask questions, and receive directed help and feedback.

Registering for a session includes 16 classes of live teaching, with each class 2 hours long. There are also assignments, readings, and guided projects supplementing each class, provided below. Each class is held from 5:30pm to 7:30pm PST.
