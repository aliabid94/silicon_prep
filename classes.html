Classes

Silcon Prep classes are all held online with a live instructor. Students are provided a link where they can watch the instructor, ask questions, and receive directed help and feedback.

Registering for a session includes 14 classes of live teaching, with each class 2 hours long. There are also assignments, readings, and guided projects supplementing each class, provided below. Each class is held from 5:30pm to 7:30pm PST.

1
The Foundations of Machine Learning: Finding Patterns in Data

We start by getting our terms straight: what is the difference between artificial intelligence, machine learning, and deep learning and do they present competing philosophies? We discuss parametric learning, and talk about common issues encountered when doing machine learning, from over-fitting to fairness and reproducibility. Along the way, each student will set up their first machine learning experiments using python notebooks.

Coding Exercise 1: Data Analysis with Tensorflow

2
From "Machine Learning" to "Deep Learning"

We quickly progress to deep learning: what are the factors that have led to the deep learning revolution? We understand how neural networks work, derive backpropagation intuitively, and discuss different types of neural networks. We observe the recent results that deep learning has achieved on a variety of datasets.

3
Convolutional Neural Networks: Helping Machines See

Next, we focus on a kind of neural network popular for machine vision related tasks, convolutional neural networks. We discuss why convolutions make sense for visual tasks, and define filter sizes, strides, and other hyperparameters used in the architecture of convolutional neural networks. We show how a neural network can be quickly trained for image classification, and discuss other applications both within computer vision and beyond.

Project 1: Shirts or Shoes? Building a Neural Network to Recognize Clothing Items

4
Tricks to Train Convolutional Networks

Training deep convolutional neural networks can be tricky, and recently, a variety of techniques have been proposed to improve training, from better optimizers to batch normalization. We will review these methods and demonstrate how they can be applied successfully in practice.

5
Computer Vision Beyond Image Recognition

There's more to computer vision than just image recognition. We'll look at other tasks, such as image segmentation and figure out how to adapt convolutional neural networks to effectively carry out these tasks.

Project 2: Smile! Training a Neural Network to Detect Facial Expression.

6
Interpreting Neural Networks & Fooling Them

Despite the power of neural networks, they have certain limitations as well. One is the vulnerability of neural networks to small "adversarial" perturbations that fool the neural network into making incorrect predictions. Another is the limited ability to interpret/understand what a neural network has learned. We'll discuss these problems, as well as how to address them.

7
From Images to Words: What Makes Language Difficult?

We will shift from images to words, and correspondingly from convolutional neural networks to recurrent neural networks. We'll review the key tasks in natural language processing and investigate the performance of the recurrent neural network architecture for these tasks.

Coding Exercise 2: Gradients, Saliency Maps and Adversarial Attacks

8
Word Embeddings and Recurrent Neural Networks: Helping Machines Read

We will spend some time in this class on word embeddings, which are a way to transform discrete words into continuous vectors. We will also dive more deeply into the recurrent neural network architectures and discuss popular flavors such as the LSTM and Transformer Architecture.

9
Text Classification with LSTMs

Using the LSTM architecture, we will discuss how to build text classification systems, such as sentiment classifiers. The focus of the class will be on models that take as input sequences and output single classes.

Project 3: Shakespeare vs. Poe: Who Wrote That?

10
Text Translation with Sequence-Sequence Models

Using an adaptation of the LSTM architecture, we will discuss how to build text manipulation systems, such as translation models. The focus of the class will be on models that take as input sequences and output sequences.

11
Unsupervised Learning: Compressing with Autoencoders

This class, we begin the final unit of the class, which covers unsupervised learning. We review classical unsupervised learning techniques such as PCA, but quickly move to unsupervised learning in the age of deep learning.

Coding Exercise 4: Using Autoencoders to Visualize High-Dimensional Data

12
Adding Probability to the Mix: Generative Models and the Variational Autoencoders

We focus on a particular kind of unsupervised learning that has recently attracted a lot of attention: generative models. Generative models are unique, as they allow machine learning systems to generate new kinds of samples, such as new images, text, and even music! We discuss how these algorithms work, and implement an important class of generative models known as variational autoencoders.

13
When Neural Networks Compete: Generative Adversarial Networks

The most popular kind of generative model is the generative adversarial network (GAN) -- we discuss what the motivation behind the GAN, and show how to implement one. GANs are difficult to train, so we discuss recent advances that make it possible to train GANs more effectively and with exciting results.

Project 4: Image Translation with GANs

14
The Future of Deep Learning (Are the Robots Coming?)

We end this class by discussing the very cutting-edge of deep learning: what are deep learning researchers working on right now, and where might they go next? We'll also discuss some project ideas and resources that the student can consult for future learning and growth.
