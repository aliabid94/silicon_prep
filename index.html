Master Machine Learning from Silicon Prep.

Machine learning has become a core technology in computer science, robotics, medical sciences, data analysis, user experience, marketing — the mass of applications have created an enormous market demand for students and engineers with machine learning competence.

However, machine learning education has long been inaccessible and difficult. We’re Stanford machine learning researchers that have developed a full curriculum involving classes and projects for beginners to guide them to excel in this field.

We teach online classes in small groups to anyone with a background in software engineering, ranging from talented high-school students to working professionals. The course consists of 16 sessions over an 8 week period. See the class schedule to view course materials.
